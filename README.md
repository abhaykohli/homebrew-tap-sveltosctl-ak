# Abhaykohli Tap-sveltosctl-ak

## How do I install these formulae?

`brew install abhaykohli/tap-sveltosctl-ak/<formula>`

Or `brew tap abhaykohli/tap-sveltosctl-ak` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "abhaykohli/tap-sveltosctl-ak"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
